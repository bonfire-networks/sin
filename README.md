# Sin

A convenient isomorphic alternative to elixir's AST. Describes
elixir syntax as structs.


## Installation

If using `mix`, add `sin` to your dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:sin, git: "https://github.com/bonfire-networks/sin", branch: "main"}
  ]
end
```
